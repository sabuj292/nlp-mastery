# 🧠 Day 1 – Introduction to NLP

## 🔍 What is NLP?
NLP (Natural Language Processing) is a field of AI and linguistics focused on enabling computers to understand, interpret, and generate human language.

## 🎯 Real-World Applications
- ChatGPT
- Google Translate
- Alexa/Siri
- Gmail Smart Reply
- Social Media Sentiment Analysis

## 📈 NLP Eras
| Era | Approach |
|-----|----------|
| Classical | Rule-Based |
| Statistical | N-grams, HMM, CRF |
| Deep Learning | RNNs, LSTMs |
| Transformers | BERT, GPT |
| LLMs/GenAI | Few-shot, Zero-shot, reasoning |
